# Randonneuring.org web application

Randonneuring.org provides several randonneuring event management tools intended to
be used by the RBA, Event Organizer, or webmaster of a region. 
When you enter your events, routes, and rosters into randonneuring.org, you get a complete automated system for managing your 
brevets and generating a variety of outputs, all automatically. These include

- High Quality PDF Cue Sheets (landscape and portrait)
- Brevet Cards in several formats
- Waivers with event details and rider names pre-printed
- Web Pages you can link to
- JSON data you can incorporate into your site
- CSV results for RUSA upload

To get up to speed quickly with these tools. 
Read the RBA/Organizer [Quick Start Guide](https://randonneuring.org/about/quick_start). For a more comprehensive description 
of the features available at randonneuring.org, read the complete documentation at [About Randonneuring.org](https://randonneuring.org/about) page.

## eBrevet Support 
When you put your events into randonneuring.org, you automatically support the [eBrevet](https://github.com/ctnadovich/ebrevet#ebrevet) mobile phone app for rides, including
a Control Check-In status web page for your use. Although randonneuring.org is not required for using eBrevet, most regions will find it to be the easiest way to do so.
